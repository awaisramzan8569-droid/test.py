# Artificial Intelligence

Artificial Intelligence:
It is a technology of computer science which learns and thinks like humans.

## Types of AI:
There are seven types of AI.

### Narrow AI:
It is a type of AI which can perform only specific task.  
It is the most common AI which is used in the world.

### General AI:
It is a type of AI which can think and learn like humans and perform different tasks like humans.  
It can’t exist yet.

### Super AI:
It is a type of AI which can be more intelligent than humans.  
It can be smarter than the human brain.  
It can make better decisions than humans.  
It is a future idea.

### Reactive Machine:
It is a type of AI which can react to current conditions.  
It can’t have memory.

### Limited Memory AI:
It is a type of AI which can use past data to make decisions.  
It has limited memory.

### Theory of Mind AI:
It is a type of AI which can understand human emotions.

### Self Aware AI:
It is a type of AI which can be aware of itself and its existence.

## Role of AI Engineer:
An AI engineer understands the problem and converts it into AI solutions, developing and deploying them into real-world applications.  
He can build a model, train it, test it, and deploy it into real-world applications.  
He can monitor his model at any time.  
He can collect data, clean data, and prepare data for models.
